Projet n°1

Captures d'écran

API-REST - WS-VidMed

Contexte : Projet personnalisé encadré qui a été réalisé dans le cadre d'un enseignement informatique, il a ensuite été évalué pour l'examen final du BTS SIO.

C'est une API-REST JSON développée à l'aide du Framework PHP CodeIgniter en version 3.1.11. On peut y exécuter des requêtes HTTP : GET/POST/PUT/DELETE sur les ressources Médecin, Médicament et Rapport de visite.

Projet n°2

Captures d'écran

Appli Android - VidMed-Android

Contexte : Projet personnalisé encadré qui a été réalisé dans le cadre d'un enseignement informatique, il a ensuite été évalué pour l'examen final du BTS SIO.

C'est une application Android développée en Java qui communique avec une API-REST JSON. Elle permet entre autres de visualiser, modifier et ajouter des médecins, médicaments et rapports de visite selon leur département sous forme de liste déroulante.

COMPÉTENCES
  • Java
  • API JSON

Projet n°3

Application web de gestion

Contexte : Stage de 2ème année de BTS SIO, réalisé chez une association prestratrice de services (gardes d'enfant et ménages principalement) : La Maison des Chaudoudoux

C'est une application de gestion développée sans framework ayant une architecture MVC. Elle permet de gérer les candidats et leurs admissions, les fiches des intervenants et leurs planning ainsi que les familles.
